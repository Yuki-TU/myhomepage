# Feature

シンプルなデザインで自分のことが紹介できるウェブサイトを構築しました。
天気予報を入れたのは、APIを使用する練習はもちろんですが、一日の始まりは天気からと私は思っており、
毎日欠かさずに見ているものなので、今日の天気を知れるようにしました。
